About StackKnot

A way for Software Developers to sharpen their skills. StackKnot allows you to unravel the proverbial knot of full stack software development. We accomplish this by making card sets to study in a flashcard interface or create different kinds of tests to study with. Also, do you dread the whiteboard? We have a section for that!. Set with questions and picture answers for Javascript and Java!
